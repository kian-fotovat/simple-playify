Music Controls

Command

Description

/play

Plays songs by title or link from various platforms. See supported platforms.

/search

Searches for a song and lets you choose from the top results.

/play-files

Plays one or more uploaded audio or video files directly.

/pause

Pauses the current song.

/resume

Resumes the paused song.

/skip

Skips to the next song in the queue.

/stop

Stops playback and disconnects the bot.

/nowplaying

Shows details about the currently playing song.

/seek

Jumps to a specific timestamp in the current song.

Queue Management

Command

Description

/queue

Displays the current song queue with interactive pages.

/playnext

Adds a song to the top of the queue.

/shuffle

Randomizes the order of songs in the queue.

/remove

Removes a specific song from the queue.

/clearqueue

Clears all songs from the queue.

Settings & Utilities

Command

Description

/filter

Applies audio effects like slowed, reverb, or bassboost.

/loop

Toggles looping for the current song.

/autoplay

Toggles automatic playback of similar songs when the queue ends.

/lyrics

Displays the lyrics for the current song.

/status

Shows the bot's performance and diagnostic stats.
