Music Controls

Command

Description

/play [query]

Play a link or search for a song. Supports various platforms.

/search [query]

Searches for a song and lets you choose from the top results.

/pause

Pauses the current playback.

/resume

Resumes the paused playback.

/skip

Skips to the next song in the queue.

/previous

Plays the previous song in the history.

/stop

Stops playback and disconnects the bot.

/nowplaying

Shows the currently playing song.

/seek

Opens an interactive menu to rewind, fast-forward, or jump to a specific time.

Queue Management

Command

Description

/queue

Displays the current queue of songs with interactive pages.

/playnext

Adds a song by link/search or a local file to the front of the queue.

/shuffle

Shuffles the current queue randomly.

/remove

Opens an interactive menu to remove one or more songs from the queue.

/jumpto

Opens a menu to jump to a specific song in the queue.

/clearqueue

Clears all songs from the queue.

Playback Modes

Command

Description

/loop

Toggles looping for the current song.

/autoplay

Enables/disables automatic playback of similar songs when the queue ends.

/24_7 [mode]

Keeps the bot in the channel. Modes: normal (loops queue), auto (adds similar songs), or off.

Settings & Utilities

Command

Description

/volume [level]

MOD

Adjusts the music volume for everyone (0-200%).

/reconnect

Refreshes the voice connection to fix lag without losing the queue.

/status

Displays the bot's full performance and diagnostic stats.

/setup controller

ADMIN

Sets a channel for the persistent music controller.
