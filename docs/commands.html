Music Controls

Command

Description

/play [query]

Plays songs by title or link from various platforms. See supported platforms

/search [query]

Searches for a song and lets you choose from the top results.

/play-files

Plays one or more uploaded audio or video files directly.

/pause

Pauses the current song.

/resume

Resumes the paused song.

/stop

Stops playback and disconnects the bot.

/nowplaying

Shows the currently playing song.

/skip

Skips to the next song in the queue. Replays the current song if loop is enabled.

Queue Management

Command

Description

/queue

Displays the current queue of songs with interactive pages.

/playnext

Adds a song by link/search or a local file to the front of the queue.

/shuffle

Shuffles the current queue randomly.

/remove

Opens an interactive menu to remove one or more songs from the queue.

/clearqueue

Clears all songs from the queue.

Playback Modes

Command

Description

/loop

Toggles looping for the current song.

/autoplay

Enables/disables automatic playback of similar songs when the queue ends.

/24_7 [mode]

Keeps the bot in the channel. Modes: `normal` (loops queue), `auto` (adds similar songs), or `off`.

Lyrics & Karaoke

Command

Description

/lyrics

Displays the lyrics of the current song.

/karaoke

Starts a karaoke session with synced lyrics. For best results, add `topic` or `audio` to your search.

Settings & Utilities

Command

Description

/filter

Applies audio filters (e.g., slowed, nightcore, bassboost) in real time.

/reconnect

Refreshes the voice connection to fix lag or audio issues without losing the queue.

/status

Displays the bot's full performance and diagnostic stats.

/discord

Get an invite to the official Playify community and support server.

/kaomoji

ADMIN

Toggles kawaii mode for fun, cute bot responses.
