Basic Controls

Command

Description

/play [query]

Plays a song, playlist, or searches by title from YouTube (including YouTube Music links), Spotify, or SoundCloud.

/pause

Pauses the current song.

/resume

Resumes the paused song.

/stop

Stops playback and disconnects the bot.

Queue Management

Command

Description

/skip

Skips to the next song in the queue.

/shuffle

Shuffles the current queue randomly.

/loop

Toggles looping for the current song.

Settings & Preferences

Command

Description

/kaomoji

ADMIN

Toggles kawaii mode for fun responses.

/language

ADMIN

Sets the bot's language (English/French) for the server.
