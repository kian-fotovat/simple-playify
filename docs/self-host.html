Simple Windows Installation Guide

1

Download the Installer

Click the download button on this page to get the `Playify_Setup.exe` file. It's a standard Windows installer, so it's safe and easy to use.

2

Run the Setup Wizard

Once downloaded, run the installer. It will install Playify on your computer and automatically launch the configuration wizard when it's finished.

3

Follow the Guided Configuration

The app will guide you through getting the necessary API keys (like your Discord Bot Token). It provides videos and links directly in the app, so no coding is required.

4

Invite & Enjoy YouTube!

At the end of the setup, you'll get a unique invite link for your bot. Use it to add the bot to your Discord server, and you're ready to play music from YouTube!

Frequently Asked Questions

Is the self-hosted version also free?

Yes, absolutely. Playify is 100% free and open-source. The only "cost" is that it runs on your own computer's resources, which are typically minimal.

What are the system requirements?

Any modern computer running Windows 10 or Windows 11 will work perfectly. The application is lightweight and doesn't require a powerful machine.

Do I need to keep my computer on for the bot to work?

Yes. Since the bot runs on your computer, it must be on for the bot to be online. To make this seamless, the app includes a "Start with Windows" option (enabled by default during setup), so you don't have to manually open it every time you restart.

Where can I get help if I'm stuck during setup?

The best place for support is our official Discord server. We have a dedicated support channel where the community and developers can help you out.
